# Rick and Morty Wiki (Astana Hub Grant Task)

### Описание
Веб-приложение для поиска персонажей, локаций и эпизодов. Проект демонстрирует навыки создания Fullstack-приложения с использованием прокси-сервера.

### Почему выбран этот стек?
- **Python (FastAPI):** Идеален для высокопроизводительных API и интеграции с AI/LLM моделями.
- **Vanilla JS & Tailwind:** Для чистоты кода, отсутствия лишних библиотек и быстрой отрисовки интерфейса.

### Как запустить проект
1. **Backend:** - Перейдите в `cd backend`
   - Установите зависимости: `pip install -r requirements.txt`
   - Запустите сервер: `python -m uvicorn main:app --reload`
2. **Frontend:** - Просто откройте `frontend/index.html` в браузере при запущенном бэкенде.

### Проектирование и компромиссы
- **Архитектура:** Использован паттерн "Proxy Server". Фронтенд не общается с внешним API напрямую, что соответствует требованиям Уровня 2.
- **Компромисс:** Для реализации роутинга вместо сложных библиотек (типа React Router) использована логика смены состояний через JavaScript (SPA-подход), что позволило уложиться в дедлайн без потери качества.
- **Известные ошибки:** В текущей версии отсутствует кэширование на бэкенде.

### Уникальные подходы
Реализован механизм **Debouncing** для поиска, чтобы не перегружать сервер запросами при каждом нажатии клавиши.
